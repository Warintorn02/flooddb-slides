บทเรียนและข้อเสนอแนะ

🔬 แยกปัญหาเป็นชั้น

Network → Auth → Render → UX การแก้ทีละชั้นพร้อมวัดผลป้องกัน regression และหา root cause ได้เร็วขึ้น

📱 ทดสอบบนเครื่องจริงเสมอ

Android WebView มีพฤติกรรมต่างจาก emulator ปัญหาหลายอย่างไม่ปรากฏบนเดสก์ท็อป

📋 เอกสารที่ดีประหยัดเวลา

บันทึก SHA256, session log, และ heartbeat ทุกรอบ ทำให้กลับมาทำต่อได้เร็วแม้เปลี่ยนเซสชัน

🛡 Page-local > Global override

การแก้เฉพาะบริบท (page-local guard) ลด side effect ดีกว่าการ override global ที่ส่งผลทั่วแอป
